# BrambleAI Landing

Marketing site for BrambleAI built with Vite, React, TypeScript, Tailwind, and shadcn-ui.

## Development

```sh
# install deps
npm install

# start dev server
npm run dev

# build for production
npm run build
```

## Notes

- Icons and touch assets live in `public/`.
- Routes are defined in `src/App.tsx`; individual pages live in `src/pages/`.
